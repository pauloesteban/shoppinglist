Paso a paso para registrar un producto.

Para realizar el Registro de un producto se debe ingresar con un usuariio predeterminado de uso exclusivo del administrador, el cuál tendrá el acceso a información de productos y clientes.

Una vez ingresado el usuario y contraseña del administrador, se visualiza la siguiente ventana, con diferentes botones que cumplen una función en específico, en este caso damos un clic en el botón Productos.

La siguiente ventan muestra diferentes ambientes de información de las categorías de los productos, cantidad de productos almacenados, y la ventana de agregar productos.

Para agregar el producto se debe seleccionar una categoría del producto que se va almacenar, una vez seleccionado se puede visualizar los productos que ya estan agregados en esa misma categoría, así como también el código del nuevo producto.

Se debe llenar todos los campos del producto, teniendo en cuenta que si falta alguno, no se podrá realizar el almacenamiento, si se realiza todo de forma correcta luego damos un clic en Registrar, el programa mostrará un mensaje de registro, que indicará que ha sido registrado un nuevo producto.
